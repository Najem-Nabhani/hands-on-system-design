## Product stages
### Stage 1
1. A simple terminal-based like/dislike counter for a single user. The user can press `l` to like, `d` to dislike, or `c` to clear their vote, and `q` to quit. 
And in return, the counter will show the current state of the counter.

#### Code stages
Basic commad-based Kotlin application.

#### Knowledge
* Setup: git init + sdkman (java 21.0.11-tem), IDEA for Maven + Kotlin.
* Kotlin: 
  * "companion object" for static-like behavior.
  * "private accessor" for encapsulation.
